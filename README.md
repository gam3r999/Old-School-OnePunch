# Old‑School OnePunch

**This is a mod for _Old School_ that gives you a one‑punch ability.**  
⚠️ **WARNING**: This is for _Old School_ only — **not** for _Wrestling Empire_ or any other game.

---

## About  
Old‑School OnePunch allows your character to dispatch opponents with a single devastating punch. Use responsibly. It’s for fun, not for other games.

---

## Requirements  
- The mod is built for _Old School_ only.  
- You must install **BepInExPack** before using this mod.  
  Download/install from: [Thunderstore — Wrestling Empire / BepInExPack](https://thunderstore.io/c/wrestling-empire/p/BepInEx/BepInExPack/)  
- Ensure the version of BepInEx (and any dependency) matches what the mod expects.

---

## Installation  
1. Download and extract the latest release of Old‑School OnePunch.  
2. Install BepInExPack (via Thunderstore) following its instructions.  
3. Place the mod files (dlls, config, etc) into your game’s `BepInEx/plugins` folder (or as per BepInExPack instructions).  
4. Launch the game. Confirm the mod is loaded (look for “OnePunch” or “Old‑School OnePunch” in the BepInEx log).  
5. Start playing and enjoy that one‑punch power!
